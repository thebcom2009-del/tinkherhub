## 🌟 Key Features

### 🛍️ Smart Shopping Cart
- A fully functional, real-time shopping cart.
- Dynamic badge count on the cart icon.
- Sliding sidebar for easy item management (add/remove/update quantities).
- Auto-calculated totals with budget-friendly formatting.

### 📋 Deep Product Insights & Reviews
- **Detail Modals**: View high-resolution images and full descriptions for every product.
- **Review System**: Read honest customer feedback and ratings.
- **Interactive Ratings**: Shoppers can submit their own star ratings and reviews instantly.

### 🎨 Premium "Amber & Earth" Aesthetic
- A sophisticated **yellow and brown** theme using an attractive mesh gradient.
- Modern **glassmorphic** navigation and UI elements.
- Unique "Nude & Earth" color palette for a calm, organic shopping experience.
- Fully responsive design that looks stunning on mobile, tablet, and desktop.

## 🛠️ Technology Stack
- **HTML5**: Semantic structure for SEO and accessibility.
- **CSS3**: Advanced animations, mesh gradients, and glassmorphism.
- **JavaScript (Vanilla)**: Dynamic product rendering, state-managed cart, and modal logic.
- **Google Fonts**: Custom typography using *Playfair Display*, *Outfit*, and *Inter*.

## 🚀 How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/thebcom2009-del/tinkherhub.git
   ```
2. Navigate to the folder:
   ```bash
   cd tinkherhub
   ```
3. Open `index.html` in your favorite browser.
